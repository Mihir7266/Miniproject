# Image Upload Guide for Admin Panel

## How to Add Images to Menu Items

### Option 1: Enter Image URL (Easiest)

1. Open Admin Panel
2. Click "Add Menu Item"
3. In the image section, you'll see an input field: **"Or enter image URL"**
4. You can use:
   - Unsplash images (recommended)
   - Any public image URL
   - Your own hosted images

**Example Unsplash URLs:**
```
https://images.unsplash.com/photo-1546069901-ba9599a7e63c?w=500&h=500&fit=crop
https://images.unsplash.com/photo-1565299624946-b28f40a0ae38?w=500
https://images.unsplash.com/photo-1563379091339-6a21f28d3e7b?w=500
```

### Option 2: Upload Local File (Currently uses placeholder)

When you upload a file using the "Choose File" button, it will show a preview, but currently uses a placeholder image from Unsplash.

### How to Get Food Images

**1. From Unsplash (Free, High Quality)**
1. Go to: https://unsplash.com/s/photos/indian-food
2. Click on any food image
3. Click "Download free"
4. Copy the direct image URL
5. Use it in your form

**Example formats:**
- Best for food: `https://images.unsplash.com/photo-XXXXX?w=500`
- Add `&h=500&fit=crop` for square images

**2. Search for specific foods:**
- Paneer: https://unsplash.com/s/photos/paneer
- Biryani: https://unsplash.com/s/photos/biryani
- Dosa: https://unsplash.com/s/photos/dosa
- Lassi: https://unsplash.com/s/photos/lassi

### Quick Tips

✅ **Square images work best** (500x500px)
✅ **Use Unsplash** - it's free and professional
✅ **Copy direct image URLs** - they're permanent
✅ **Test URLs** before saving by pasting in browser

### Features

- **Live Preview**: See image before saving
- **URL Input**: Fast and easy
- **File Upload**: For future use
- **Professional Look**: Images make menu more attractive

## Example Workflow

1. Go to Admin Panel
2. Click "Add Menu Item"
3. Fill basic info (name, description, price)
4. For image:
   - Open Unsplash in new tab
   - Search for food item
   - Copy image URL
   - Paste in "Or enter image URL" field
5. Check the preview shows
6. Complete form and save

Your menu items will now have beautiful images on the public menu!
